Este proyecto realiza un análisis comparativo de pacientes con COVID-19 en México (2021 vs 2023) utilizando K-Means y Fuzzy C-Means para identificar perfiles clínicos y niveles de gravedad. Se emplean variables demográficas, clínicas y comorbilidades a partir de los datos abiertos de la Secretaría de Salud.

Carpetas del repositorio:
- Datos/ – enlace a la fuente oficial
- Tablas y Figuras/ – resultados generados
- ProyectoAnalítica.ipynb – notebook con el análisis (K-means y C-means)
